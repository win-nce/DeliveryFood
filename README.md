# Ресторан
> Проверка NodeJs
 ```bash
node -v
```

> Проверка NPM
 ```bash
npm -v
```

> Создать React проект **Vite**
 ```bash
npm create vite@latest
```

- Название проекта
- Выбираем фреймворк: React
- Язык программирования: TypeScript

>Установка зависимых пакетов
```bash
npm i
```
 
 >Запуск проекта [localhost5173] (http://localhost:5173)
```bash
npm run dev
```
